Noella Wang - Mutator

Project Details:

Date: August 2025

Role: Independent Developer (Max, gen~)

Repo: View on GitHub

Mutator is a multi-sequencing effector Max for Live device developed in Max/MSP. It treats audio processing as something rhythmic and interactive, rather than a static insert on a channel.

It features six audio effects in sequential order: pitch shifter, beat repeat, flanger, phaser, downsampler, and Schroeder reverb, each implemented as compact DSP patches designed for clarity and performance reliability.

The device's core is its sequencer, which triggers each effect's on/off state. It automatically syncs its step count to Ableton's time signature. You can set the step length (1/4, 1/8, 1/16) with dotted or triplet rhythms over one- or two-bar cycles. Each effect has one main parameter (scaled 0-100) for easy control, plus a global dry/wet mix. The UI also features a dynamic color palette that selects from 12 presets on load.
